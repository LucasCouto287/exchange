Project: CURRENCY EXCHANGE | Angular 8.2.8 | Bootstrap 4

How you can use: 
Step 1) Dowonload the project
Step 2) Angular requires Node.js version 10.9.0 or later. install node ==> https://nodejs.org/en/
Step 3) Install npm (in project folder)
Step 4) Install bootstrap (in project folder)  ==> npm install bootstrap --save
Step 5) Run the application (in project folder)  ==> ng serve 
Step 6) Open your browser => http://localhost:4200/
